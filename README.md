# 📦 Supply Chain Performance Dashboard (Power BI)

## 🔎 Project Overview

This project is an interactive **Supply Chain Performance Dashboard**
built using **Power BI** to analyze profitability, supplier performance,
transportation efficiency, and product quality.

The dashboard enables business stakeholders to monitor KPIs and make
data-driven decisions to improve operational efficiency and reduce
defects.

## 🎯 Business Objectives

-   Identify the most profitable supply routes\
-   Detect the best and worst-performing suppliers\
-   Analyze defect rates across transportation modes\
-   Monitor lead time and operational efficiency\
-   Provide actionable insights using visual analytics

------------------------------------------------------------------------

## 📊 Key Performance Indicators (KPIs)

-   💰 **Total Profit**
-   📦 **Total Orders**
-   🚚 **Best Route (Highest Profit)**
-   🏆 **Best Supplier (Lowest Defect Rate)**
-   ⚠️ **Worst Transportation Mode (Highest Defect Rate)**
-   ⏳ **Average Lead Time**
